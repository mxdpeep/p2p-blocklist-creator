# P2P Blocklist Builder

GitHub: [https://github.com/mxdpeep/p2p-blocklist-creator]

Downloads and builds P2P blocklist for your torrenting client.

## Usage

* **make** -> CLI help

You may simply use the pre-created blocklist from this address:
